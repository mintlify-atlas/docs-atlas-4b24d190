Welcome to the official documentation for the **Universal Robots ROS2 Driver** — a production-ready bridge between ROS2 and the complete UR cobot family, from the 3 kg UR3 to the 30 kg UR30. Built on `ros2_control` and the Universal_Robots_Client_Library, it brings real-time joint control, force/torque control, and MoveIt2 integration to every supported UR robot.

## Get up and running

<Steps>
  <Step title="Install the driver">
    Install from ROS binary packages with a single `apt` command, or build from source for custom workflows.
  </Step>
  <Step title="Configure your robot">
    Install the External Control URCap on your teach pendant, set the host IP, and extract your robot's kinematics calibration.
  </Step>
  <Step title="Launch the driver">
    ```bash
    ros2 launch ur_robot_driver ur_control.launch.py ur_type:=ur5e robot_ip:=192.168.56.101
    ```
  </Step>
  <Step title="Start moving">
    Activate a controller and send trajectory commands, or open RViz and use MoveIt2 for full motion planning.
  </Step>
</Steps>
